# Playground Server

## Requirements

- Linux or OS X (not working in Windows)
- python 2.7

## How to work server

```
$ cd /path/to/libzen
$ make #make libzen.jar

$ cd /path/to/libzen/playgroundserver
$ sudo easy_install bottle
$ python server.py

```

Then, you can use Playground in http://localhost:3000 .
